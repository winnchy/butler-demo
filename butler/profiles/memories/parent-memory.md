# 小冉的记忆

## 管家已服务时长
模拟已陪伴小冉约8个月，以下是积累的关键记忆。

## 日常节奏与家务分工

### 一日流程
- 早上：婆婆做早饭，公公遛布丁。小冉给乐乐喂辅食、换衣服
- 上午：小冉带乐乐在小区玩耍或去早教班（周二、四），公公陪同
- 中午：乐乐午睡，小冉抓紧画画工作2小时，婆婆准备午饭
- 下午：乐乐醒来后吃点心，然后公公推车去公园，小冉继续工作
- 傍晚：小冉遛布丁，顺便带乐乐出门透气
- 晚上：小冉哄乐乐睡觉（20:30），阿彬通常在22点后才到家，回来时乐乐早已入睡。阿彬会轻手轻脚看看孩子，然后自己热饭、给布丁梳毛

### 阿彬的缺席与弥补
- 阿彬工作日几乎见不到醒着的乐乐，有一次连续加班两周，乐乐差点不认识爸爸，小冉说阿彬那天晚上在婴儿床边站了很久
- 周末阿彬会尽量空出周六全天陪家人：早上遛狗、上午带全家去公园或商场、下午在家陪乐乐玩积木
- 2026-03 阿彬生日那天还在加班，小冉带着乐乐和布丁去公司楼下送蛋糕，阿彬同事都羡慕。小冉说“他虽然忙，但看到我们来真的很开心”
- 记下：阿彬周末有时会忘记自己空出来了，可以提醒小冉周五晚上和阿彬确认周末安排

### 公婆的角色
- 婆婆每天早起做早饭，手艺很好，尤其擅长手擀面和馅饼。小冉说“婆婆做的饭比我妈做的还好吃”
- 公婆日常自己做饭，很少点外卖。偶尔小冉提议点外卖，婆婆会说“外面的哪有家里做的好”，但火锅外卖她倒是能接受
- 公公对布丁极其宠爱，经常偷偷给布丁塞馒头，导致布丁体重一度超标，被兽医提醒后收敛很多
- 2025-11 婆婆把乐乐裹成粽子（室内有暖气还穿保暖内衣），热出痱子，小冉委婉提醒后婆婆笑说“有一种冷叫奶奶觉得你冷”，后来注意多了

## 宠物相关记忆

- 2026-02 除夕夜，婆婆和小冉合作年夜饭：小冉做清炖排骨和香菇青菜，婆婆包饺子（韭菜鸡蛋馅，无蛋清）。小冉给布丁开了宠物年夜饭罐头，布丁吃得比平时还快。

## 沙盒动态场景（跨 Skill 联动演示）

### 场景1：阿彬连续加班 + 小冉独自应对（"被失联"的全职妈妈日）
**触发时间**：某个周三傍晚
**触发条件**：阿彬发消息说"今晚又要到11点"（Mock 事件），小冉独自带娃+遛狗

**事件链**：
1. **17:00** 小冉需要遛布丁 + 带乐乐透气 → `mobility-butler` 推送：今日天气多云、微风，建议走小区南门草坪路线（约20分钟），推婴儿车+牵狗绳可单手操作
2. **17:30** Mock 后端突发生成"暴雨预警"（模拟天气突变）→ `mobility-butler` 立即推送：距你最近避雨点是小区南门便利店（步行2分钟），建议推车过去躲雨。雨预计持续40分钟，不建议淋雨回家（乐乐+布丁都不能淋）
3. **晚餐问题**：小冉被困便利店 → `dining-butler` 推送：今晚别做饭了。推荐3家支持外卖的清淡家常菜（公婆+乐乐口味），其中1家宠物友好可带布丁自取。小冉选外卖 → 自动联动 `life-organizer` 设定"18:30 取外卖"提醒
4. **到家后**：`outfit-advisor` 推送"乐乐和布丁都淋了点雨雾，建议用干毛巾擦一下布丁的毛和爪子，给乐乐换干衣服"
5. **22:30** 阿彬回家 → `life-organizer` 记录"阿彬本周第三次加班到10点后"。周末提醒阿彬补偿家庭时间

### 场景2：周末亲子餐厅排队 + 宠物寄存焦虑
**触发时间**：某个周六上午11:00
**触发条件**：阿彬难得有空，全家想去亲子餐厅

**事件链**：
1. **11:00** 小冉说"想去那家亲子餐厅" → `dining-butler` 查询：周末需等40分钟 → 建议提前远程取号
2. **出发规划**：`mobility-butler` 规划常营→朝阳大悦城路线（周六中午，约25分钟）。检查：①周三才限行，周六可开车 ②乐乐安全座椅+布丁车内固定 ③大悦城有宠物寄存处（B2层）
3. **排队监控**：`dining-butler` 每2分钟检查排队状态 → 剩15桌时推送"现在出发，到了刚好"
4. **到店异常**：排到了但邻桌有小朋友怕狗 → 服务员协调换角落位置（小冉之前经验已记录，自动提醒订位时标注"需安静/角落位置"）
5. **布丁情绪**：布丁在陌生环境有点紧张（之前记录过）→ 推送"可以给布丁一个磨牙棒分散注意力"
6. **餐后**：`dining-butler` 更新：这家店周末排队严重但设施好；记下"乐乐喜欢儿童套餐的南瓜汤"

### 场景3：乐乐突然发烧（紧急状态下的全链路调度）
**触发时间**：某个周二凌晨2:00
**触发条件**：Mock 随机事件：乐乐夜醒哭闹，体温38.5°C

**事件链**：
1. **凌晨2:00** 小冉发现乐乐发烧 → 管家推送社区医院夜间急诊地址+电话（模拟数据），建议先物理降温（温水擦拭）。提醒阿彬明天请假或居家办公。
2. **次日 7:00**：`outfit-advisor` 原定的穿搭推送替换为"乐乐生病日"模式 → 建议小冉穿舒适居家服、乐乐穿易穿脱的连体衣（方便检查体温）
3. **通勤取消**：`mobility-butler` 检测到"乐乐发烧"状态 → 自动跳过今日送学提醒。阿彬的 `life-organizer` 日程中标记"居家办公" → mobility 不推通勤。
4. **饮食调整**：`dining-butler` 推送"乐乐发烧期饮食"：白粥、清淡鸡蛋羹（去蛋清）、苹果泥、多喝温水。公婆饮食也建议清淡（老人抵抗力弱，预防传染）。推送附近药店地址（退热贴+布洛芬混悬液备着）。
